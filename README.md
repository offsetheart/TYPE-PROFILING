# TYPE-PROFILING

## This program is tool for searching and profiling user's type and design user's tour character

### Main Function
1. AI Travel Profiling- Analyzes users’ travel behavior patterns and generates a personalized TripCode based on travel style, destination preference, food preference, planning tendency, and crowd tolerance.
2. Personalized Travel Experience Design- Combines TripCode, transportation method, tourism density data, and Korea Tourism Organization data to recommend optimized travel districts and personalized travel patterns.
